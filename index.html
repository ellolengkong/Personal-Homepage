About

A little bit about me

Halo! Saya Marchello Andreas Lengkong, yang akrab disapa Ello atau Cello. Saat ini, saya adalah mahasiswa semester 4 Program Studi Teknik Informatika di Universitas Sam Ratulangi (UNSRAT). Berdomisili di Kembes, Sulawesi Utara, saya membangun website ini sebagai bagian dari tugas kuliah sekaligus wadah untuk mengekspresikan diri melalui Personal Homepage.

Saya memiliki ketertarikan besar pada pengembangan web dan desain antarmuka (UI/UX). Saat ini, saya fokus mendalami struktur teknologi web seperti HTML dan juga CSS dan JavaScript untuk menciptakan halaman yang rapi, terorganisir, dan fungsional.

Di luar dunia perkodingan, saya adalah seorang pencinta film dan seorang gamer yang menikmati berbagai jenis permainan. Akhir-akhir ini, saya sedang banyak menghabiskan waktu di game bergenre Sandbox maupun MOBA. Selain itu, saya memiliki rasa ingin tahu yang tinggi untuk mempelajari hal-hal baru dan merupakan seorang rubik enthusiast. Di samping kesibukan akademik, saya juga aktif dalam kegiatan gereja, khususnya dalam lomba Cerdas Cermat Alkitab (CCA), di mana saya bersyukur telah meraih beberapa prestasi di bidang tersebut.
